# Level milestones (main track)

Curve: `level = floor(14 * ln(1 + total_xp / 120)) + 1` (no cap).

| Level | Meaning |
|------:|---------|
| 1 | Start |
| 5 | Fundamentals stick |
| 10 | Small projects with guidance |
| 15 | Independent snippets |
| 20 | Feature owner (strong junior / early mid) |
| 25 | Design-aware before coding |
| 30 | Reviewer mindset |
| 35 | Mid/senior path |
| 40 | System thinker |
| 45 | Cross-domain comfort |
| 50 | Staff-depth judgment |
| 60 | High-quality OSS contributor |
| 70 | **Solo OSS builder** — substantial projects alone |
| 80 | Field-shaping reliability under ambiguity |
| 90 | Research↔eng bridge |
| 100 | **World-class rare** (top-hundred territory) |

Levels past 100 continue on the same curve; rarity increases, not a hard title gate.

## How hard should XP be?

Meaningful session with real practice: often ~15–80 XP.
Exceptional independent design + implementation + tests: up to ~150–220 (cap per award).
Chat-only conceptual help: 0–15 XP.

Prioritize **transfer and independence** over raw line count.
